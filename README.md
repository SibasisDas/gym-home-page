# 🧮 BMI Calculator

One of the main features of this project is the **BMI (Body Mass Index) Calculator**.

### Features

- Users can enter their height and weight.
- Calculates BMI instantly using JavaScript.
- Displays the calculated BMI value.
- Shows the BMI category.
- User-friendly interface.
- Responsive design.

### BMI Categories

| BMI Range | Category |
|-----------|----------|
| Below 18.5 | Underweight |
| 18.5 – 24.9 | Normal Weight |
| 25.0 – 29.9 | Overweight |
| 30.0 and Above | Obese |

### BMI Formula

```
BMI = Weight (kg) ÷ (Height (m) × Height (m))
```

Example

Weight = 70 kg

Height = 1.75 m

BMI = 70 ÷ (1.75 × 1.75)

BMI = **22.86**

Category = **Normal Weight**
